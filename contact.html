Realtime Chat

Realtime Chat with Firebase

This is a sample implementation of Realtime Chat feature available at Firebase. Using onAuthStateChanged(), user token availability at browser is checked. If there is no user token available, user is asked to sign in with various sign In options.

Per Log-In completion, browser will output Log-In Successful alert. New User activity continues in sandbox.
